Temperature Converter

You’ll often need to determine which part of a program is run based on user input or other events. Create a program that converts temperatures from Fahren- heit to Celsius or from Celsius to Fahrenheit. Prompt for the starting temperature. The program should prompt for the type of conversion and then perform the conversion. The formulas are C = (F − 32) × 5 / 9 and F = (C × 9 / 5) + 32
